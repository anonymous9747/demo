An improved multi-band parallel generation model based on MelGAN for better and faster waveform generation

Abstract: In this paper we propose an improved model (iMelGAN) based on MelGAN, we prove that iMelGAN is able to produce higher quality speech than MelGAN, and is capable of generating audio that is 5 times faster than real time on CPU when using Pytorch inference. At the same time, we propose a multi-band parallel generation model based on iMelGAN. The proposed Multi-band iMelGAN, which has only 1.91M model parameters, effectively reduces the total computational complexity from 7.6 GFLOPS to 0.95 GFLOPS, which enables the model to generate audio that is 35 times faster than real time on CPU when using Pytorch inference without optimization. This makes it easier to deploy speech synthesis applications on lower-power devices
